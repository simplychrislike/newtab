# newTab
newTab is an extension for Google Chrome.
It replaces the original "new tab" with a blank page.

## Available options

### Change background color
The options page allows you to customize the background color of the blank new
tab.

## Future options
- Load a new background image every hour/day from various sources.
- Adding your own background image
- Set random background color
